collective.encrypted Installation
---------------------------------

To install collective.encrypted using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``collective.encrypted`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        collective.encrypted
       
* Re-run buildout, e.g. with:

    $ ./bin/buildout
